# Datell Palette Catalog

Use this catalog as the source-of-truth list of built-in Datell palette IDs that the current report builder exposes.

## Rules

- Use the palette ID verbatim.
- Keep chart series, KPI emphasis, table heat states, and severity cues tied to palette variables.
- Preserve the current light-versus-dark behavior: light palettes may use a gradient header, dark palettes should keep a high-contrast solid header.

## Business

- `palette-classic`: professional blue default.
- `palette-royal-blue`: premium royal-blue corporate palette.
- `palette-crimson-biz`: bold corporate-red palette.
- `palette-amber-gold`: warm amber-gold business palette.
- `palette-teal-corp`: teal business palette.
